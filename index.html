Backend beállítás

Ez a szavazás egy ingyenes JSONBin.io tárolóban őrzi a válaszokat, így mindenki ugyanoda ír.

Regisztrálj a jsonbin.io oldalon → hozz létre egy új bint {} tartalommal → illeszd be a Bin azonosítót és a Master kulcsot alul.

A bin URL-jéből vagy az info panelből

Fiókból → API kulcsok

Ezek az adatok csak a te böngésződ localStorage-ában tárolódnak — sehova máshova nem kerülnek, csak közvetlenül a JSONBin-hez. Ha inkább bekódolnád, szerkeszd a CONFIG blokkot a HTML fájl tetején.
